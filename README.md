# Ejercicio Página Web

Este proyecto consiste en la creación de una página web sobre animales en peligro de extinción.

## Contenido
- Página principal con carrusel de imágenes
- Subpáginas de mamíferos y aves
- Video informativo
- Formulario
- Navegación entre páginas

## Objetivo
Aplicar conocimientos de desarrollo web mediante la creación de una página interactiva.

## Nota
Esta información fue elaborada con fines académicos, las imágenes y videos son responsabilidad de los propietarios.
